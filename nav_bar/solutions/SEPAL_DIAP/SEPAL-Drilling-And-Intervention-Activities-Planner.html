Overview

Proactive Coordination of well activity

SEPAL Drilling and Intervention Activities Planner is an end-to-end well activity scheduling platform, embedded with a robust set of features that tackle data uncertainty, improve data veracity and availability, reduce data dependency delays, enhance activity scheduling, rig optimisation, compliance monitoring, reporting and cross departmental collaboration. With all information integrated in a single platform, engineers and managers can always access updated well activity schedules, with functionalities to drill down to the dependencies and detailed information underlaying each activity and the evolution of schedules. SEPAL DIAP can be deployed on-premise, on the cloud or as a hybrid model.

SEPAL DIAP promotes the following:

  • Improved cross-functional collaboration resulting in robust schedule planning and decision-making.
  • Enhanced efficiency in terms of increased response time to project changes.
  • Seamless projects swap to accommodate changes due to delay or accelerated.
  • Reduced complexities in creating, updating, and managing project schedules, budget templates and reports.
  • Increased efficiency via data visualization and analytics.
  • Enhanced transparency and speed of decision making.
  • High-fidelity database for rigs and wells.
  • Enhanced data management and security.

SEPAL DIAP enables proactive coordination of all activity in the drilling sequence, including critical pre-work tasks, company-wide rig schedule, drilling activities as well as completions and intervention (including rigless) activities.

The solution works on any device (and any operating system) connected to the internet anywhere anytime, enabling enhanced transparency and speed of decision making. It promotes increased virtual teamwork such as global teams and real-time collaboration

SEPAL DIAP provides a high-fidelity database for rigs and wells, and a single-source-of-truth for well and rig information.

SEPAL DIAP has powerful visualisation features such as charts and reports which aids on-the-fly decision making.

SEPAL DIAP supports easy attachment of critical documents, such as well schematics, forecast results, permits, and AFEs to the well schedule. It also support budget template creation and documentation, daily drilling report documentation, and AAR report documentation.

SEPAL DIAP is a web-based application that gives users uninterrupted access to the application. No downloads, installs, license keys, service packs or compatibility issues - everyone has the latest version, and quick access to quality-checked information. It offers a notification system that keeps the users up to date with changes in any sequence.

SEPAL DIAP gives the application admin access to the application log, to effectively track all changes in the application. This is this ensure data integrity in the application.

The solution has a highly scalable interface that both novice and experienced software users can get the hang of in little to no-time.
